# Radio

## Import

`import { Radio } from 'fannypack'`

## Basic Usage

<Playground>
  <Radio name="weather" label="Sunny" value="sunny" />
  <Radio name="weather" label="Windy" value="windy" />
</Playground>

## Disabled

Make the Radio disabled with the `disabled` prop.

<Playground>
  <Radio disabled name="hasFannypack" label="Yes" value={true} />
  <Radio checked disabled name="hasFannypack" label="No" value={false} />
</Playground>

## States

A Radio can use different states (as per palette) such as `danger`, `success` and `warning`.

<Playground>
  <Radio state="danger" name="weather" label="Sunny" value="sunny" />
  <Radio state="success" name="weather" label="Rainy" value="rainy" />
  <Radio state="warning" name="weather" label="Windy" value="windy" />
  <Radio state="primary" name="weather" label="Overcast" value="overcast" />
</Playground>

## Default checked

To check an option by default, use the `defaultChecked` prop.

<Playground>
  <Radio name="weather" label="Sunny" value="sunny" />
  <Radio defaultChecked name="weather" label="Rainy" value="rainy" />
  <Radio name="weather" label="Windy" value="windy" />
  <Radio name="weather" label="Overcast" value="overcast" />
</Playground>
