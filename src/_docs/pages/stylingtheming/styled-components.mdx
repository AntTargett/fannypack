# Styled Components

Fannypack uses [styled-components](https://www.styled-components.com/docs/basics#getting-started) internally to style components with a `styled` function.

Here is a basic example using `styled` with **Fannypack**:

```jsx
import { Button, styled } from 'fannypack';

const RedButton = styled(Button)`
  background-color: red;
  color: white;
`;

<RedButton>I am a red button!</RedButton>
```
