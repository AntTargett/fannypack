## Lists

To group a set of buttons, use the `<Tags>` component.

<Playground>
  <Tags>
    <Tag>Hello</Tag>
    <Tag>World</Tag>
    <Tag>This</Tag>
    <Tag>Is</Tag>
    <Tag>Me</Tag>
  </Tags>
</Playground>

## Grouping

You can group tags to make one tag with the `<Group>` component.

<Playground>
  <Group>
    <Tag>Default</Tag>
    <Tag palette="textTint">Hello</Tag>
  </Group>
</Playground>
