## Prerequisites

To install Medikit Components, you must have **react@16.x** and **styled-components@3.x** installed as a dependency to your project.

To install these prerequisit packages, run the following command:

```curl
npm install react@latest styled-components@latest
```

## Installing the components

Run the following command to install **Medikit Components**:

```curl
npm install medikit
```

## Getting set up

To start using the components, please follow these steps:

1. Wrap your application in a `<ThemeProvider>` which is provided by **Medikit Components**:

```jsx
import { ThemeProvider } from 'medikit';

const App = () => (
  <ThemeProvider>
    // ... your app
  </ThemeProvider>
);
```

2. Now you can start using components like so!:

```jsx
import { Button } from 'medikit';

const MyApp = () => (
  <Button>
    Hello world!
  </Button>
);
```
