import { Playground, PropsTable } from 'docz';
import { Button, Buttons } from '../index';

## Import

`import { Button, Buttons } from 'medikit'`

## Basic Usage

<Playground>
  <Button>Default button</Button>
</Playground>

## Custom elements

Button, by default, is actually wrapped in a `div` element. See more [here](https://reakit.io/components/button).
However, you are able to change it to a custom element using the `as` prop.

<Playground>
  <Buttons>
    <Button>Default button</Button>
    <Button as="a">Anchor button</Button>
  </Buttons>
</Playground>

## Colors

Colors can be added to a button with the `color` prop.

<Playground>
  <Buttons>
    <Button>Default button</Button>
    <Button color="primary">Primary button</Button>
    <Button color="success">Success button</Button>
    <Button color="danger">Danger button</Button>
    <Button color="warning">Warning button</Button>
  </Buttons>
</Playground>

## Loading

To indicate a button is loading, use the `isLoading` prop.

<Playground>
  <Buttons>
    <Button isLoading>Default button</Button>
    <Button color="primary" isLoading>Primary button</Button>
    <Button color="primary" isLink isLoading>Primary button</Button>
    <Button color="success" isOutlined isLoading>Success button</Button>
  </Buttons>
</Playground>

## Outlines

An outline can be placed on a button using the `isOutlined` prop.

<Playground>
  <Buttons>
    <Button color="primary" isOutlined>Primary button</Button>
    <Button color="success" isOutlined>Success button</Button>
    <Button color="danger" isOutlined>Danger button</Button>
  </Buttons>
</Playground>

## Sizes

Buttons are available in four different sizes and can be added with a `size` prop.

<Playground>
  <Buttons>
    <Button size="small">Small button</Button>
    <Button>Default button</Button>
    <Button size="medium">Medium button</Button>
    <Button size="large">Large button</Button>
  </Buttons>
</Playground>

## Links

A button can appear as a link with no styling by adding the `isLink` prop.

<Playground>
  <Button isLink color="primary">Default button</Button>
</Playground>
