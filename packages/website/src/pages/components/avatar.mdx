# Avatar

## Import

```jsx
import { Avatar } from 'fannypack';
```

## Usage

```jsx-live
<Avatar src="/bean.jpg" alt="Bean" />
```

### Shapes

Avatars are available in two different shapes and can be changed using the `kind` prop.

```jsx-live
<Avatar kind="circle" src="/bean.jpg" alt="Bean" />
```

### Sizes

Avatars are available in five different sizes or you can set your own using the `size` prop.

```jsx-live
<Set>
  <Avatar kind="circle" src="/bean.jpg" alt="Bean" size="small" />
  <Avatar kind="circle" src="/bean.jpg" alt="Bean" />
  <Avatar kind="circle" src="/bean.jpg" alt="Bean" size="medium" />
  <Avatar kind="circle" src="/bean.jpg" alt="Bean" size="large" />
  <Avatar kind="circle" src="/bean.jpg" alt="Bean" size="150px" />
</Set>
```

### Initials

Avatars can also be initials using the `initials` prop.

```jsx-live
<Set>
  <Avatar initials="JM" kind="circle" palette="primary" size="small" />
  <Avatar initials="Jake Moxey" kind="circle" palette="secondary" />
  <Avatar initials="JM" kind="circle" palette="success" size="medium" />
  <Avatar initials="JM" kind="circle" palette="warning" size="large" />
</Set>
```
