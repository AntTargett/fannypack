# Icon

## Import

```js
import { Icon } from 'fannypack';
```

## Usage

```jsx-live
<Icon icon="solid-igloo" />
```

### Sizes

Change the size of an icon with the `fontSize` prop.

```jsx-live
<Icon a11yLabel="Igloo" icon="solid-igloo" fontSize="100" />
<Icon a11yLabel="Igloo" icon="solid-igloo" marginLeft="major-1" />
<Icon a11yLabel="Igloo" icon="solid-igloo" fontSize="300" marginLeft="major-1" />
<Icon a11yLabel="Igloo" icon="solid-igloo" fontSize="400" marginLeft="major-1" />
<Icon a11yLabel="Igloo" icon="solid-igloo" fontSize="500" marginLeft="major-1" />
<Icon a11yLabel="Igloo" icon="solid-igloo" fontSize="600" marginLeft="major-1" />
<Icon a11yLabel="Igloo" icon="solid-igloo" fontSize="700" marginLeft="major-1" />
<Icon a11yLabel="Igloo" icon="solid-igloo" fontSize="800" marginLeft="major-1" />
<Icon a11yLabel="Igloo" icon="solid-igloo" fontSize="900" marginLeft="major-1" />
```

### Colors

Change the color of an icon with the `color` prop. It can be any color from the [palette](/palette), or any other color like `#ff0000` or `red`.

```jsx-live
<Icon a11yLabel="Success" icon="success" color="success" />
<Icon a11yLabel="Error" icon="danger" color="danger" marginLeft="major-1" />
<Icon a11yLabel="Warning" icon="warning" color="warning" marginLeft="major-1" />
<Icon a11yLabel="Copy" icon="solid-clipboard" color="primary" marginLeft="major-1" />
```

### Usage in text

Icons will be the same size as its text by default.

```jsx-live
<Text>
  This is text with an icon! <Icon a11yLabel="Thumbs up" icon="solid-thumbs-up" />
</Text>
<Heading use="h3" marginTop="major-2">
  This is a heading with an icon! <Icon a11yLabel="Thumbs up" icon="solid-thumbs-up" />
</Heading>
```
