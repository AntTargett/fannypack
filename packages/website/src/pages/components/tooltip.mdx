# Tooltip

## Import

```jsx
import { Tooltip } from 'fannypack';
```

## Usage

```function-live
function Example() {
  const tooltip = Tooltip.useState();

  return (
    <Box>
      <Tooltip.Reference {...tooltip} use={Button}>
        Reference
      </Tooltip.Reference>
      <Tooltip {...tooltip}>
        What was the best thing that happened to you today?
      </Tooltip>
    </Box>
  )
}
```
