# Dialog

## Import

```jsx
import { Dialog } from 'fannypack';
```

## Usage

```jsx-live
<Dialog title="This is a title" width="600px">
  Lorem ipsum dolor sit amet, consectetur adipiscing elit. Suspendisse diam ipsum, cursus id placerat congue, ultrices eget lectus. Duis posuere, lacus sed tristique commodo, sapien turpis mollis nunc, vestibulum consectetur lectus augue sit amet justo.
</Dialog>
```

### Footer

You can render a footer for the dialog using the `footer` prop.

```jsx-live
<Dialog
  title="This is a title"
  footer={<Text>This is a footer</Text>}
  width="600px"
>
  Lorem ipsum dolor sit amet, consectetur adipiscing elit. Suspendisse diam ipsum, cursus id placerat congue, ultrices
  eget lectus. Duis posuere, lacus sed tristique commodo, sapien turpis mollis nunc, vestibulum consectetur lectus
  augue sit amet justo.
</Dialog>
```

### Action buttons

It is common to have action buttons in a dialog's footer. Use the `showActionButtons` prop to enable them. Dialog also accepts the `actionButtonsProps` that accepts the props of the [`<ActionButtons>` component](/components/button#actionbuttons-props).

```jsx-live
<Dialog
  actionButtonsProps={{ onClickSubmit: () => alert('Submitted') }}
  title="This is a title"
  showActionButtons
  width="600px"
>
  Lorem ipsum dolor sit amet, consectetur adipiscing elit. Suspendisse diam ipsum, cursus id placerat congue, ultrices
  eget lectus. Duis posuere, lacus sed tristique commodo, sapien turpis mollis nunc, vestibulum consectetur lectus
  augue sit amet justo.
</Dialog>
```

### Close button

To render a close icon on a dialog, use the `showCloseButton` prop.

```jsx-live
<Dialog
  onClickClose={() => console.log('close')}
  showCloseButton
  title="This is a title"
  width="600px"
>
  Lorem ipsum dolor sit amet, consectetur adipiscing elit. Suspendisse diam ipsum, cursus id placerat congue, ultrices
  eget lectus. Duis posuere, lacus sed tristique commodo, sapien turpis mollis nunc, vestibulum consectetur lectus
  augue sit amet justo.
</Dialog>
```

#### Initial focus

```function-live
function Example() {
  const modal = Modal.useState();
  const closeButtonRef = React.useRef();

  return (
    <div>
      <Modal.Disclosure {...modal}>Open modal</Modal.Disclosure>
      <Modal.Backdrop {...modal} />
      <Dialog.Modal
        unstable_initialFocusRef={closeButtonRef}
        title="This is a title"
        showCloseButton
        {...modal}
      >
        Lorem ipsum dolor sit amet, consectetur adipiscing elit. Suspendisse diam ipsum, cursus id placerat congue, ultrices
        eget lectus. Duis posuere, lacus sed tristique commodo, sapien turpis mollis nunc, vestibulum consectetur lectus
        augue sit amet justo.
        <Button ref={closeButtonRef} onClick={modal.hide}>Close</Button>
      </Dialog.Modal>
    </div>
  )
}
```
