# FieldWrapper

### Import

```jsx
import { FieldWrapper } from 'fannypack'
```

### Basic Usage

The `<FieldWrapper>` component is a utility belt that wraps around form elements.

```.jsx
<FieldWrapper a11yId="username" label="Username">
  <Input />
</FieldWrapper>
```

### Tooltips

You can show a tooltip by supplying a string or an element to the `tooltip` prop.

```.jsx
<FieldWrapper a11yId="username" label="Username" tooltip="Your username must be awesome.">
  <Input />
</FieldWrapper>
```

```.jsx
<FieldWrapper
  a11yId="username"
  label="Username"
  tooltip={
    <Text fontSize="150">Your username must be <Text fontWeight="bold">awesome.</Text></Text>
  }
>
  <Input />
</FieldWrapper>
```

#### Custom tooltip trigger

You can modify the tooltip trigger button by supplying an element to the `tooltipTrigger` prop.

```.jsx
<FieldWrapper
  a11yId="username"
  label="Username"
  tooltip="Your username must be awesome."
  tooltipTrigger={<Button size="small" minHeight="unset">Tooltip</Button>}
>
  <Input />
</FieldWrapper>
```

Or you can set a default trigger via `defaultProps` [in the theme](#theming)

#### Custom tooltip popover props

You can modify the tooltip popover props by supplying `tooltipPopoverProps`.

```.jsx
<FieldWrapper
  a11yId="username"
  label="Username"
  tooltip="Your username must be awesome."
  tooltipPopoverProps={{ placement: 'top-start' }}
>
  <Input />
</FieldWrapper>
```

Or you can set a default popover props via `defaultProps` [in the theme](#theming)

### Descriptions

The description text is placed underneath the label.

```.jsx
<FieldWrapper a11yId="username" label="Username" description="Required for your fannypack">
  <Input />
</FieldWrapper>
```

### Hints

The hint text is placed underneath the field child.

```.jsx
<FieldWrapper a11yId="username" label="Username" hint="Must be awesome">
  <Input />
</FieldWrapper>
```

### Optional fields

To indicate a field as optional, add the `isOptional` prop.

```.jsx
<FieldWrapper a11yId="username" label="Username" isOptional>
  <Input />
</FieldWrapper>
```

### Required fields

```.jsx
<FieldWrapper a11yId="username" label="Username" isRequired>
  <Input />
</FieldWrapper>
```

### States

```.jsx
<FieldWrapper
  a11yId="username"
  label="Username"
  validationText="This is an invalid name"
  state="danger"
>
  <Input defaultValue="Jake" />
</FieldWrapper>
<FieldWrapper
  a11yId="fannypacks"
  validationText="Please check the box!"
  state="primary"
  marginTop="major-3"
>
  <Checkbox label="Fannypacks are great" />
</FieldWrapper>
<FieldWrapper
  a11yId="accept"
  label="Do you accept?"
  validationText="Nice one!"
  state="success"
  marginTop="major-3"
>
  <RadioGroup
    defaultValue={true}
    name="accept"
    options={[{ label: 'Yes', value: true }, { label: 'No', value: false }]}
  />
</FieldWrapper>
```
